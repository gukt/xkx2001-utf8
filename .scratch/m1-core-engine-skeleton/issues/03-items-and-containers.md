# 03 — 物品与容器:拾取/丢弃/物品栏

**What to build:** 引入物品实体与一个通用的"物品容器"组件——房间地面与玩家物品栏本质是同一种能力("这个实体持有一堆物品"),不做成两个专属命名的组件,而是同一种容器组件,房间实体与玩家实体各挂一份。新增三个命令:

- `take <物品>`:把当前房间地面容器里的物品移到玩家物品栏容器。
- `drop <物品>`:把玩家物品栏容器里的物品移到当前房间地面容器。
- `inventory`/`i`:列出玩家物品栏容器当前持有的全部物品。

物品名称匹配复用 02 号票的通用目标别名匹配工具(物品可以有别名,不必输入完全规范名称)。尝试拾取/丢弃一个不存在的物品(不在对应容器里)收到明确提示,不崩溃。

物品本身需要展示文本(简述 + 详细描述),复用 01 号票房间已有的"描述"组件形状(若物品的展示文本需求与房间一致,应合并成一个通用描述组件供房间与物品共用,不各自专属)。

**Blocked by:** 02 — 复用其目标别名匹配工具与"解析 -> 意图 -> 执行"管线来接入这三个新命令。
