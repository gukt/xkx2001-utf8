## 问题

对「题材无关引擎 + UGC 创作者组合实体」而言，ECS 相比继承树 / Feature 混入，
在哪些 UGC 硬场景上更顺手、在哪些场景上更别扭？

旧结论（仅作对照）：「ECS 在 Python 是架构选择非性能优化」——本原型不测性能，
只摸 **创作者意图 → 运行时行为** 的路径摩擦。
